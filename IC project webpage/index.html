Internet Radio working with Microchip

Internet radio services allows user to stream different kinds of audio and transfer those digital signal to analog signal so as to allow user to listen different kinds of media. In this process, media will be compressed into smaller size and sent to the hardware immediately so as to allow user to listen and switch the channels immediately. Compare with downloading audios, it does not consume a lot of time for the users that it won’t waste some time for buffering audio. However, since it is an internet radio, it can’t be stopped and replay since it is a real-time broadcasting station

Exposure

By using UV Light LED, students can print and fabricate the PCB board by place the printed transparency. After pumping out the vaccum and exposure it under the LED Light, the PCB board needs to perform development so as to display the copper remains on the PCB Board.

Developing

Developing is a chemical processes in PCB Fabrication, it is used to define the circuit's appereance when the PCB fabricationis finished.
Developing is a simple process that it is easy to overlook with the production of high quality PCB’ and to overlook the details necessary to produce high quality developing.
